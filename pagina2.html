Modelo Entidad Relación

Es una metodología para diseñar y representar bases de datos de una manera conceptual, se usa para estructurar

la información de un sistema de entidades, atributos y relaciones

Componentes principales del MER

Entidades: estas representas objetos u conceptos que tienen existencia en el mundo real que tienen una existencia

independiente a la base de datos, se clasifica en:

-Entidades fuertes: Son una Fuente totalmente independiente como Cliente y Proveedor

-Entidades débiles: Depende de otra entidad para existir como “Detalles de factura” depende de “factura”
